# 📉 股票

## 市场类型说明

所有股票接口均需传入 `type` 参数指定市场类型：

| type | 市场 |
|------|------|
| `0` | 深交所及其他 |
| `1` | 上交所 |
| `2` | 未知 |
| `116` | 港股 |
| `105` | 美股 |
| `155` | 英股 |

## 获取股票分时走势图

**路由**：`/stockTrends2`

返回股票当日或近几日的分时走势，含每分钟的价格、均价和成交量，用于绘制分时图。

**参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | string | ✅ | 市场类型（见上表） |
| `code` | string | ✅ | 股票代码，例：`600519` |
| `ndays` | number | | 天数：`1`=今日（默认）  `5`=近5日 |

**示例** `/stockTrends2?type=1&code=600519&ndays=1`

```json
{
  "code": "600519",
  "market": 1,
  "name": "贵州茅台",
  "decimal": 2,
  "preClose": 1850.0,
  "trendsTotal": 241,
  "trends": [
    "2023-04-10 09:30,1855.00,1852.30,48392,89766400.00"
  ]
}
```

**trends 字段格式**：`时间,最新价,均价,成交量(手),成交额(元)`

## 获取股票 K 线数据

**路由**：`/stockKline`

返回指定股票在指定周期和复权方式下的 K 线数据，适用于 K 线图渲染。

**参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | string | ✅ | 市场类型 |
| `code` | string | ✅ | 股票代码 |
| `klt` | number | ✅ | K 线周期（见下表） |
| `fqt` | number | | 复权：`0`=不复权（默认）  `1`=前复权  `2`=后复权 |
| `lmt` | number | | 返回条数，默认 500 |
| `end` | string | | 截止日期，格式 `YYYYMMDD`，不传返回最新 |

**K线周期 klt**

| 值 | 周期 |
|----|------|
| `1` | 1分钟 |
| `5` | 5分钟 |
| `15` | 15分钟 |
| `30` | 30分钟 |
| `60` | 60分钟 |
| `101` | 日线 |
| `102` | 周线 |
| `103` | 月线 |

**示例** `/stockKline?type=1&code=600519&klt=101&fqt=1&lmt=100`

```json
{
  "code": "600519",
  "market": 1,
  "name": "贵州茅台",
  "decimal": 2,
  "dktotal": 7845,
  "klines": [
    "2023-04-10,1840.00,1860.00,1835.00,1855.00,48392,89766400.00"
  ]
}
```

**klines 字段格式**：`日期,开盘价,最高价,最低价,收盘价,成交量(手),成交额(元)`

## 获取股票逐笔成交明细

**路由**：`/stockDetails`

返回指定股票最近 N 笔逐笔成交记录，含成交时间、价格、成交量及买卖方向。

**参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | string | ✅ | 市场类型 |
| `code` | string | ✅ | 股票代码，例：`600519` |
| `pos` | number | | 起始位置，负数表示最后 N 笔，默认 `-14` |

**示例** `/stockDetails?type=1&code=600519`

```json
{
  "code": "600519",
  "market": 1,
  "decimal": 2,
  "prePrice": 1850.0,
  "details": [
    "14:56:42,1855.00,100,185500,2"
  ]
}
```

**details 字段格式**：`时间,价格,成交量(手),成交额(元),方向(1=买入 2=卖出 3=中性)`

## 获取股票行情快照

**路由**：`/stockGet`

返回指定股票的完整实时行情快照，含最新价、涨跌幅、成交量、市盈率、资金流向等全量字段。

**参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | string | ✅ | 市场类型 |
| `code` | string | ✅ | 股票代码，例：`600519` |

**示例** `/stockGet?type=1&code=600519`

```json
{
  "f57": "600519",
  "f58": "贵州茅台",
  "f127": "食品饮料",
  "f43": 185500,
  "f44": 186000,
  "f45": 184800,
  "f169": 500,
  "f170": 27,
  "f168": 22,
  "f47": 48392,
  "f48": 8976640000,
  "f116": 2330000000000,
  "f117": 2090000000000,
  "f162": 3195,
  "f135": 438159072,
  "f136": 394149126,
  "f137": 44009946
}
```

**常用字段说明**

| 字段 | 含义 | 说明 |
|------|------|------|
| `f57` | 股票代码 | |
| `f58` | 股票名称 | |
| `f127` | 行业 | |
| `f43` | 最新价 | ×100 |
| `f44` | 最高价 | ×100 |
| `f45` | 最低价 | ×100 |
| `f169` | 涨跌额 | ×100 |
| `f170` | 涨跌幅 | ×100，即 `f170/100` = 实际百分比 |
| `f168` | 换手率 | ×100 |
| `f47` | 成交量 | 手 |
| `f48` | 成交额 | 元 |
| `f116` | 总市值 | 元 |
| `f117` | 流通市值 | 元 |
| `f162` | 市盈率 | ×100 |
| `f135` | 主力流入 | 元 |
| `f136` | 主力流出 | 元 |
| `f137` | 主力净流向 | 元（正=净流入，负=净流出） |
