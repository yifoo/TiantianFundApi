# 📊 基金详情

## 获取基金简介

**路由**：`/fundMNStopWatch`

返回基金概要卡片信息，适合列表项或搜索结果展示。

**参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `FCODE` | string | ✅ | 基金代码，例：`003834` |

**示例** `/fundMNStopWatch?FCODE=003834`

```json
{

    "FCODE": "003834",

    "JJGS": "华夏基金",
    "DWJZ": "3.114",
```

## 获取各阶段涨跌幅

**路由**：`/fundMNPeriodIncrease`

返回基金在标准时间维度下的历史收益率（近1周/月/季/半年/1-3年/今年以来/成立以来）。

**参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `FCODE` | string | ✅ | 基金代码 |
| `RANGE` | string | | 范围：`n`=年度（`3n`=3年）  `y`=月度（`3y`=3月）  不填=全阶段 |

**示例** `/fundMNPeriodIncrease?FCODE=003834`

```json
{
  "Datas": [{
    "title": "近1年",
    "syl": "43.73",
    "avg": "7.90",
    "hs300": "-5.20",
    "rank": "26",
    "sc": "748"
  }]
}
```

|------|------|------|------|
| `FCODE` | string | ✅ | 基金代码 |
| `RANGE` | string | | 同 `/fundMNPeriodIncrease` 的 `RANGE` |

```json

    "QSC": "748"
  }]

```

## 获取基金持仓股票及实时行情

**路由**：`/getStockRatio`

两步聚合接口：先获取基金持仓列表，再批量查询每只持仓股的实时行情。行情接口失败时降级返回基础持仓数据。

**参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `FCODE` | string | ✅ | 基金代码，例：`003834` |

**示例** `/getStockRatio?FCODE=003834`

```json
{
  "code": 200,
  "data": [{
    "GPDM": "600438",
    "GPJC": "通威股份",
    "JZBL": "8.32",
    "MKTCAP": "1532640000",
    "NEWTEXCH": "1",
    "price": 42.56,
    "ratio": 1.23
  }]
}
```

::: tip 注意
`price` 和 `ratio` 字段为实时行情，若行情接口请求失败则这两个字段不存在，其余持仓字段仍会正常返回。
:::
