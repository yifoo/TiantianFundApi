| 模块 | 接口数 | 说明 |

| [📊 基金详情](./funddetail.md) | 9 | 基金简介、详情、评级、持仓、费率等 |
| [💰 基金净值](./nav.md) | 8 | 历史净值、实时估值、走势图数据 |

| [🎯 基金筛选](./select.md) | 3 | 多维度条件筛选 + 筛选配置接口 |
| [🌐 基金主题](./theme.md) | 4 | 主题列表、热门主题、焦点主题 |
| [🏢 基金公司](./company.md) | 5 | 公司列表、详情、指数等 |
| [👤 基金经理](./manager.md) | 8 | 经理信息、业绩、风格、持仓等 |
| [📈 大数据榜单](./bigdata.md) | 2 | 大数据榜单列表与详情 |

| [📉 股票](./stock.md) | 4 | 股票详情、分时、K线、逐笔成交 |

## 通用说明

### 请求格式

所有接口均通过 `GET` 请求，参数以 Query String 形式传递：

```
GET http://localhost:3000/{路由名}?参数1=值1&参数2=值2
```

### 响应格式

大部分接口返回如下结构：

```json
{
  "code": 200,
  "msg": "获取成功",
  "data": { ... }
}
```

部分接口直接透传上游原始响应，字段结构以各接口说明为准。

### 错误码

| code | 含义 |
|------|------|
| 200 | 成功 |
| 400 | 请求失败（参数错误或上游返回异常） |
| 500 | 服务器内部错误 |
