# ⚡ 实时行情

## 获取市场指数行情快照

**路由**：`/marketIndex`

返回指定证券代码的实时行情数据，秒级更新，适用于首页行情看板。

**参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `secids` | string | ✅ | 指数代码列表，格式 `市场.代码`，多个逗号分隔（上交所前缀 `1`，深交所前缀 `0`） |
| `fields` | string | | 返回字段，不传返回全量 |

**常用指数 secids**

| secids | 名称 |
|--------|------|
| `1.000001` | 上证指数 |
| `0.399001` | 深证成指 |
| `0.399006` | 创业板指 |
| `1.000300` | 沪深300 |
| `1.000016` | 上证50 |
| `1.000905` | 中证500 |

**示例** `/marketIndex?secids=1.000001,0.399001,0.399006`

```json
{
  "code": 200,
  "msg": "获取指数成功",
  "data": {
    "diff": [{
      "f2": 316852,
      "f3": 29,
      "f4": 891,
      "f12": "000001",
      "f14": "上证指数"
    }]
  }
}
```

**常用 fields 字段说明**

| 字段 | 含义 |
|------|------|
| `f2` | 最新价（×100） |
| `f3` | 涨跌幅(%)（×100） |
| `f4` | 涨跌额（×100） |
| `f5` | 成交量（手） |
| `f6` | 成交额（元） |
| `f12` | 证券代码 |
| `f14` | 证券名称 |

## 订阅实时行情推送（SSE）

**路由**：`/sse`

建立 Server-Sent Events 长连接，持续接收东方财富行情推送，适合实时行情看板。连接建立后服务端持续推送，客户端断开时连接自动关闭。

**参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `secids` | string | ✅ | 行情代码列表，格式同 `/marketIndex` |
| `fields` | string | | 订阅字段，例：`f2,f3,f12,f14` |

**示例（浏览器端）**

```javascript
const es = new EventSource(
  'http://localhost:3000/sse?secids=1.000001,0.399001&fields=f2,f3,f12,f14'
)

es.onmessage = (e) => {
  const data = JSON.parse(e.data)
  console.log(data)
}

// 关闭连接
// es.close()
```

**推送数据格式**

```json
{
  "data": {
    "diff": [{
      "f2": 316852,
      "f3": 29,
      "f12": "000001",
      "f14": "上证指数"
    }]
  }
}
```

::: tip 注意
SSE 接口会保持长连接，请确保在不需要时调用 `es.close()` 释放连接。
:::

## 判断当日是否为交易日

**路由**：`/guessInfo`

通过小倍养基接口返回当日交易日状态及估算日期配置，适合前端判断是否展示盘中估值组件。

**参数**：无

**示例** `/guessInfo`

```json
{
  "data": {
    "isTradingDay": true,
    "valuationDate": "2024-03-08",
    "navDate": "2024-03-07"
  }
}
```

## 持仓股票自选净值（天天基金版）

**路由**：`/FundCoreDiyNew`

批量获取自选基金的持仓净值及涨跌幅，返回基金名称、单位净值、累计净值、日涨幅、申购状态等。

**参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `FCODES` | string | ✅ | 基金代码列表，逗号分隔，例：`003834,001975` |

**示例** `/FundCoreDiyNew?FCODES=003834,001975`

```json
{
  "data": [{
    "FCODE": "003834",
    "SHORTNAME": "华夏能源革新股票A",
    "DWJZ": "3.095",
    "LJJZ": "3.095",
    "RZDF": "0.61",
    "FSRQ": "2022-05-25",
    "ISBUY": "1",
    "FTYPE": "股票型"
  }]
}
```
